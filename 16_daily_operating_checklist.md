# Daily Operating Checklist

Designed for a student operator with limited hours.

## Research

- [ ] review 3 to 5 new leads
- [ ] verify URLs and public contact details
- [ ] add notes to CRM
- [ ] mark any uncertain facts as `unknown` or `needs verification`

## Outreach

- [ ] send or prepare 3 to 5 thoughtful outreach attempts
- [ ] personalize each first touch
- [ ] log every attempt in the outreach tracker

## Follow-Up

- [ ] check for replies
- [ ] send due follow-ups manually
- [ ] stop outreach where no further contact is appropriate

## Fulfillment

- [ ] move one client task forward
- [ ] test one live deliverable manually
- [ ] document what was changed

## Admin

- [ ] update pipeline statuses
- [ ] log revenue, proposals, and open tasks
- [ ] check next appointments and reminders

## Learning / Improvement

- [ ] capture one lesson from outreach or delivery
- [ ] improve one template, SOP, or prompt

## Minimum Viable Daily Version

If time is tight, complete:

- [ ] 2 new leads
- [ ] 1 follow-up
- [ ] 1 fulfillment task
- [ ] 5-minute KPI update
