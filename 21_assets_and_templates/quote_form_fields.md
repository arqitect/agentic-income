# Quote Form Fields

Use the minimum fields needed to qualify the lead and follow up well.

## Recommended Fields

- full name
- phone
- email
- service needed
- property type
- city / neighborhood
- address or approximate address if useful
- preferred contact method
- job details / notes
- preferred timing
- how they found the business

## Optional Fields

- budget range
- photo upload
- gate / access notes
- urgency level

## Rules

- keep mobile completion easy
- do not ask for more than is operationally useful
- tell the user what happens next
