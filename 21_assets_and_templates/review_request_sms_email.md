# Review Request SMS / Email

## SMS Template

Hi {{customer_first_name}}, thanks again for choosing {{business_name}}. If the job went well, would you mind leaving a quick review here? {{review_link}} It really helps local businesses like ours. Thank you.

## Email Template

Subject: Quick favor

Hi {{customer_first_name}},

Thanks again for working with {{business_name}}. If you were happy with the job, would you mind leaving a quick review here?

{{review_link}}

It helps a lot and only takes a minute.

Thank you,
{{sender_name}}

## Rules

- send only after service completion
- do not pressure customers
- make sure the review link is correct
- do not use this for cold outreach
- if SMS is used, confirm the client has appropriate consent and opt-out handling
- when in doubt, use manual email instead of automated text
