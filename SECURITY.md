# Security Policy

## Reporting a vulnerability

If you discover a security issue in this repository (for example, exposed secrets, unsafe automation instructions, or data-handling risks), please report it privately by opening a private security advisory on GitHub.

## What to include

- Description of the issue
- Affected file(s)
- Reproduction steps
- Potential impact
- Suggested remediation

## Response expectations

Best effort triage within 7 days and remediation planning based on severity.
